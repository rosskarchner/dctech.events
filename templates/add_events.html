Contributing to DC Tech Events

This is a community-driven calendar managed as an open source project. You can add events and (even better!) new iCal sources by forking the project on GitHub and submitting a pull request.

We're working on an easier way to contribute! In the meantime, if you have events to submit and don't want to use GitHub, feel free to email ross@karchner.com.
